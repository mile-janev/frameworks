Дијаграми

  1. Site frameworci komparacija na "Select All" akciite.
  2. Site frameworci komparacija megu findAll(Yii), fetchAll(zend), get_where(codeigniter)
  3. Site frameworci komparacija megju find(Yii), fetchRow(zend), Select one (codeigniter)
  4. Yii, komparacija megju find() i findByPk() so cel da se dokaze deka e podobro da se koristi findByPk() namesto find() koga imame primaren kluc
  5. Zend, komparacija megju find() i fetchRow() so cel da se dokaze deka e podobro da se koristi find() namesto fetchRow() koga imame primaren kluc

Функции

Yii

Методите кои започнуваат со find враќаат еден резултат, а оние со findAll враќаат низа од резултати. Секој резултат е објект кој содржи параметри со соодветните колони од табелата.

  1. Select All - findAll() - Сите записи од табелата. findAll() не прима параметри.
  2. actionFind() - find() - Еден запис со користење на find() методот. Прима објект од класата CDbCriteria како параметар.
  3. actionFindbypk() - findByPk() - Еден запис со користење на findByPk() методот. findByPk() прима цел број (примарен клуч) како параметар.
  4. actionFindbyattributes() - findByAttributes() - Еден запис со користење на findByAttributes() методот. findByAttributes() како параметар прима низа во која key е атрибутот а value е условот.
  5. actionFindbysql() - findBySql() - Еден запис со користење на findBySql() методот. Прима два параметри: првиот е query каде наместо услов се става некој стринг, а вториот е низа во која на стринговите од query-то и се доделуваат вредности. Пример, findBySql("SELECT * FROM s_post WHERE id = :id", array(":id"=>5))
  6. actionFindall() - findAll() - Повеќе записи со користење на findAll() методот. Доколку не прима параметар ги враќа сите записи од табелата. Прима параметар објект од класата CDbCriteria().
  7. actionFindallbypk() - findAllByPk() - Повеќе записи со користење на findAllByPk() методот. Прима низа од цели броеви (примарни клучеви) како параметар и во низа ги враќа соодветните редови од табелата.
  8. actionFindallbyattributes() - findAllByAttributes() - Повеќе записи со користење на findAllByAttributes() методот. Прима низа во која key е атрибутот а value е условот, а како резултат враќа низа од редови од табелата кои го исполнуваат условот.
  9. actionFindallbysql() - findAllBySql() - Повеќе записи со користење на findAllBySql() методот. Прима два параметри: првиот е query каде наместо услов се става некој стринг, а вториот е низа во која на стринговите од query-то и се доделуваат вредности. Пример, findBySql("SELECT * FROM s_post WHERE id = :id", array(":id"=>5)) Враќа низа од редови од табелата кои го исполниле условот.

Zend

  1. Select All - fetchAll() - selectallAction() - Сите записи со користење на fetchAll() методот.
  2. find() - findAction() - Еден запис со користење на find() методот, кој прима примарен клуч како параметар а враќа еден ред од база.
  3. fetchRow() - fetchrowAction() - Еден запис со користење на fetchRow() методот, кој прима атрибут и вредност на атрибутот како параметар а враќа еден ред од база.
  4. fetchAll() - fetchallAction() - Повеќе записи со користење на fetchAll() методот, кој прима атрибут и вредност на атрибут како параметар а враќа соодветни записи од база
  5. fetch() - fetchAction() - Еден запис со користење на fetch() методот на query object.

Codeigniter

  1. Select All - selectall() - get() - Сите записи со користење на get() методот.
  2. get_where() - getwhere() - Еден запис со користење на методот get_where().
  3. get() - getmethod() - Повеќе записи со користење на get() методот.
  4. query() - querymethod() - Повеќе записи со користење на custom query, односно $this->db->query("SELECT * FROM s_post WHERE id = ".$this->i);
